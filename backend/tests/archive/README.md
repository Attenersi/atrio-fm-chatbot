# Archive (No-Delete Buffer)

This folder is for historical test suites/results that are no longer active but
should be preserved for traceability.

## Subfolders

- `suites/` - old suite definitions kept as baseline snapshots
- `results/` - historical run outputs
- `sources/` - imported ad-hoc source packs

Do not write fresh run outputs here. New outputs belong in `backend/tests/results`.

# Centrale documentatiemap

Deze map bevat **kopieën** voor bundeling of offline lezen. **Wijzigingen horen in de originelen** in de repo (`docs/`, `backend/`, enz.).

**Canonieke documentatie-index**: [`../docs/documentation_map.md`](../docs/documentation_map.md)

## Deze bundel verversen (bijv. vóór een zip)

Kopieer of gelijk trekken met de repo zodat interne links kloppen:

| Bron in repo | Doel in bundel |
| --- | --- |
| `docs/*.md` | `documentation/docs/` |
| `backend/README.md`, `backend/README.nl.md` | `documentation/backend/` |
| **`backend/test_runbook.md`** | **`documentation/backend/test_runbook.md`** |

Het bestand **`documentation/docs/documentation_map.md`** is een bundelvariant van `docs/documentation_map.md` (paden aangepast). Bewerk vooral **`docs/documentation_map.md`** en werk daarna de bundelkopie bij.

Bestanden alleen in de root (`LICENSE`, `CONTRIBUTING.md`, enz.) staan niet dubbel hier; ze staan in [`../docs/documentation_map.md`](../docs/documentation_map.md).
